A simple plugin to add support for Nigerian Naira (NGN) to Gravity Forms.

== Description ==
Nigerian Naira for Gravity Forms is a basic plugin that adds currency support for Nigerians using Gravity Forms.

Nothing to configure. Just activate it and select your currency from within Gravity Forms.

== Installation ==
1. Install using the zip file in your WordPress dashboard or extracting the package into wp-content/plugins.
2. Activate the plugin.
3. Select NGN in your Gravity Forms currency settings.
